# 📄 Smart PDF Question Answering App

This is a Streamlit-based application that allows users to upload a PDF and ask questions from it using a BERT model.

## 🚀 Features

- Upload PDF
- Extract text
- Ask questions
- Get answers using BERT

## 🛠 Tech Stack

- Python
- Streamlit
- Transformers (HuggingFace)
- PyTorch
- PyPDF2

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
